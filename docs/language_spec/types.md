# Data Types
## Primitives Types
### ```int```
The ```ìnt``` type represents integers, positive or negative.
#### Exemple :
```apl
var a: int = 10;
var b: int = -3;
```
### ```float```
The ```float``` type represents floating-point numbers, positive or negative.
#### Exemple :
```apl
var pi: float = 3.14159;
var e: float = 2.71828;
```
### ```bool```
The ```bool``` type represents boolean value (```true``` or ```false```).
#### Exemple :
```apl
var isActive: bool = true;
var isCompleted: bool = false;
```
### ```string```
The ```string``` type represents a sequence of characters.
#### Exemple :
```apl
var name: string = "Alice";
var greeting: string = "Hello, world!";
```
### ```None```
The ```None``` type represents an absence of value, equivalent to null in other languages.
#### Exemple :
```apl
var result: None = None;
```
